Projet a Thyssenkrupp

Lors de mon stage j'ai pu déveloper une application en c# pour simuler le mouvement les bobines de métal dans la halle qui les stockais.



Ci-dessous le chahier des charges



Dans ce projet j'ai pu approfondir mes compétence en C# en faisant une connection TCP sur un serveur distant pour envoyer les position des bobines en temps réel.



L'application fonctionne d'une tel façon a ce que l'on puisse démarer une communication vers le serveur dès lors que l'on appuie sur le bouton Start cela lance la communication.
